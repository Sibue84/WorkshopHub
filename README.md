# WorkshopHub

Willkommen bei **WorkshopHub**!

WorkshopHub ist eine moderne Webanwendung zur Verwaltung einer Werkstatt.

## Funktionen (geplant)

- 📦 Regale verwalten
- 📦 Kisten verwalten
- 🔍 Schnelle Artikelsuche
- 🏷️ QR-Codes für Kisten
- 📷 Fotos von Kisten und Artikeln
- 🔧 Werkzeugverwaltung
- 🧵 Filamentverwaltung
- 📈 Bestandsverwaltung
- 🛒 Einkaufsliste
